# User Guide

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Wallet connection capability (MetaMask, Coinbase Wallet, etc.)
- Internet connection

### Initial Setup
1. Visit the DiversiFi application
2. Connect your wallet when prompted
3. Select your preferred cultural financial strategy
4. Review supported assets for your connected network

### Environment Variables
For development setup, copy `.env.example` to `.env.local` and add:
- `GOOGLE_AI_API_KEY` - for AI analysis
- `NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID` - for wallet connectivity

## Using the Platform

### Portfolio Management
1. **View Holdings**: See your current token balances across supported networks
2. **Select Strategy**: Choose from 7 cultural financial strategies
3. **Monitor Performance**: Track your portfolio's inflation protection

### Token Swapping
1. **Select Network**: Choose between Celo, Arbitrum, or other supported chains
2. **Pick Tokens**: Select from available stablecoins and regional currencies
3. **Enter Amount**: Specify the amount to swap
4. **Review Details**: Check exchange rates, fees, and slippage
5. **Confirm Transaction**: Approve and execute the swap via your wallet

## Supported Networks & Assets

- **Stablecoins**: cUSD, cEUR, cREAL, G$

### Arbitrum
- **Stablecoins**: USDC, USDY

## Troubleshooting

### Common Issues

#### Wallet Connection Problems
- Ensure your wallet extension is enabled
- Check that you're on the correct network
- Refresh the page if connection fails
- Try disconnecting and reconnecting your wallet

#### Swap Failures
- Verify sufficient balance for the swap amount
- Check network gas fees and ensure enough ETH/native token
- Confirm slippage tolerance settings
- Try again during lower network congestion periods

#### Network Switching
- The app automatically detects your current network
- Supported networks are displayed with available tokens
- Switch networks in your wallet application
- The interface updates automatically when network changes

### Performance Tips
- Use a stable internet connection
- Clear browser cache periodically
- Close other resource-intensive tabs
- Ensure your device has sufficient memory
