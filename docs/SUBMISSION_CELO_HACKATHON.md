# 🧠 DiversiFi: Multi-Chain AI Wealth Protection
### *Celo Hackathon V2: Build Agents for the Real World*

## 🌟 Project Overview
DiversiFi is an autonomous AI-powered financial guardian designed to protect wealth for users in volatile economies. By combining Celo's localized stablecoin ecosystem (Mento) with multi-chain RWA (Real World Asset) exposure, DiversiFi empowers users to hedge against inflation through culturally-aligned financial strategies.

**Tagline:** Protecting global wealth through culturally-aware AI Agents and Celo-native liquidity.

## 🚀 The Problem
Users in emerging markets (Africa, LatAm, SE Asia) face a "double squeeze":
1.  **Local Currency Devaluation:** Inflation rates often exceed 20-100%, erasing savings.
2.  **Complexity Barrier:** Moving into USD, Gold, or Treasury yields requires navigating complex DeFi protocols, bridge risks, and gas token management (ETH/CELO/MATIC).

## 💡 The Solution: The Autonomous Agent
DiversiFi introduces a **"Set and Forget"** wealth protection agent that:
-   **Understands Culture:** Uses 7 unique "Financial Philosophies" (e.g., *Africapitalism*, *Islamic Finance*, *Buen Vivir*) to filter assets.
-   **Automates Protection:** Monitors real-time inflation and yield spikes to rebalance assets between Celo Mento stables and Arbitrum RWAs.
-   **Solves Gas Friction:** Implements the **Agent Fuel Model**, allowing users to fund their agent with simple USDC. The agent handles all underlying gas and bridging complexity.

## 🛠️ Celo Integration ("Real World" Impact)
Celo is the "Heart" of DiversiFi's regional strategy:
-   **Mento Protocol:** We leverage Celo's unique stablecoin suite (cUSD, cEUR, cREAL, cKES, cGHS) to provide localized on-ramps and off-ramps that reflect the user's local economy.
-   **GoodDollar UBI:** Integrated G$ claiming to help users build their first "seed" capital through daily streaks.
-   **SocialConnect:** Utilizing Celo's identity layer to make sending/receiving as simple as a phone number (mapped via Libera/Valora issuers).
-   **MiniPay Native:** Optimized for the MiniPay ecosystem to reach millions of users in Africa and beyond, featuring ultra-lightweight mobile components and seamless wallet integration.
-   **AgentScan Discovery:** Implemented the Celo Agent Protocol manifest (`.well-known/agent.json`) to ensure DiversiFi is discoverable by the Celo AI Explorer and registry.
-   **Low-Cost Execution:** Celo’s sub-cent fees make high-frequency rebalancing viable for small-balance users (UBI recipients).

## 🤖 Technical Architecture
-   **Brain (Arc Agent):** An autonomous reasoning engine that monitors Envio/Celo indexers for yield spikes.
-   **Hands (OpenClaw Gateway):** A secure execution layer that signs and submits transactions to Celo/Arbitrum/Hyperliquid.
-   **Fuel (Circle MPC):** User-funded USDC sub-wallets that power the agent's "Machine-to-Machine" economy.
-   **Intelligence:** Gemini 3.0 (Portfolio analysis) + Venice AI (Private market inference).

## 📈 Track Focus: Agents for the Real World
Our submission addresses the core hackathon themes:
-   **Financial Inclusion:** Making sophisticated hedging (Gold, Treasury yields) accessible to users who only understand "local currency" and "USDC".
-   **Autonomous Operations:** The agent proactively prompts the user when it detects a currency crash, offering a one-tap "Rescue" into Celo Mento stables.
-   **UX Innovation:** The "Agent Fuel" gauge replaces the confusing "Gas/Gwei" terminology with a simple "Battery" metaphor.
