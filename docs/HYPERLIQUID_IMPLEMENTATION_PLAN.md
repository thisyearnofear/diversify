# Hyperliquid Implementation Plan (2026)

This document outlines the strategy for integrating **Hyperliquid** as a specialized "Risk Hedging Spoke" within the DiversiFi Agent architecture.

## Role in the "Hub & Spoke" Model

While **Arc L1** handles the agent's logic and data payments, **Hyperliquid** is used exclusively for:
1.  **Delta Neutral Hedging**: Automatically opening 1x short positions against user holdings (e.g., ETH, BTC) during high-volatility events.
2.  **Risk Management**: Rapidly offloading risk without needing to sell the underlying asset (tax efficiency).

### 1. Key Management (The "Agent Fuel" Approach)
Hyperliquid uses a specialized L1 setup where an "API Wallet" (signing key) is authorized to trade on behalf of a "Main Account."

*   **Main Account**: The User's Circle MPC Sub-Wallet (Agent Fuel Wallet).
*   **API Wallet**: A specialized session key generated by the Arc Agent.
*   **Flow**:
    1.  User funds their Agent Fuel Wallet on Arc L1.
    2.  Agent bridges a portion of "Fuel" (USDC) to Hyperliquid via CCTP/Arbitrum.
    3.  Agent generates an ephemeral API key pair.
    4.  Agent signs an "Approve Agent" transaction on Hyperliquid using the Circle MPC key.
    5.  **Result**: The Agent can now execute sub-second hedges using the ephemeral key, while the funds remain custodied by the Circle MPC wallet.

## Risk Management (Safety First)

1.  **Max Drawdown Limit**: The agent will never keep a hedge open if the loss exceeds X% (stop loss on the hedge itself).
2.  **Funding Rate Protection**: If funding rates turn deeply negative (costing the user money to short), the agent closes the hedge.
3.  **Isolation**: Hyperliquid sub-accounts are isolated. The agent cannot drain the user's main wallet, only the "Hedge Allocation."

## 2026 Vision: "Invisible Insurance"
The goal is for the user to see a "Capital Protected" badge on their dashboard. Behind the scenes, the Arc Agent is dynamically adjusting a short position on Hyperliquid to ensure their portfolio value (in USD terms) remains stable, even if the market crashes 20%.
